# NAME

Lazy::Utils - Utilities for lazy

# VERSION

version 1.01

# SYNOPSIS

Utilities for lazy

# INSTALLATION

To install this module type the following

        perl Makefile.PL
        make
        make test
        make install

from CPAN

        cpan -i Lazy::Utils

# DEPENDENCIES

This module requires these other modules and libraries:

- Switch
- FindBin
- Cwd
- File::Basename
